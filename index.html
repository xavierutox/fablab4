    Introducción

    La siguiente es la documentación de la API de Recursos para la cual se crearon 11 endpoints. Estos serán enumerados y explicados a continuación. Para la entrega y documentación de la API se eligió la herramienta Open Source de Slate

    Para el uso y consumo de la API en Python se utiliza Request, el cual se puede instalar con pip3 install request

    Recursos

    /Recursos/Search/:tipo_maquina

    Utilizar Request para poder consumir la información

    import requests

endpoint= 'Search'
url = f'http://3.138.175.135:4000/Recursos/{endpoint}/1'
response = requests.get(url)
print(response.json())

    Esta llamada entrega una respuesta del siguiente formato:

    {'url': 
  [{
  'url': 'https://www.youtube.com/watch?v=dQw4w9WgXcQ',
  'id': 7
  }]
}

    Devuelve todas las maquinas de un cierto tipo dado.

    HTTP Request

    GET http://3.138.175.135:4000/Recursos/Search/:tipo_maquina

    URL Parameters

    Parametros Descripción
    tipo_maquina Refiere al tipo de maquina.

    /Recursos/Add

    Utilizar Request para poder consumir la información

    
import requests

endpoint= 'Add'
url = f'http://3.138.175.135:4000/Recursos/{endpoint}'
response = requests.post(url, params={
    'tipo_maquina': '1',
    'url': 'https://www.youtube.com/watch?v=dQw4w9WgXcQ',
})
print(response.json())

    Esta llamada entrega una respuesta del siguiente formato:

    {
  'respuesta': 
    'OK', 'recurso': 
      {
        'id': 13, 
        'tipo_maquina': '1', 
        'url': 'https://www.youtube.com/watch?v=dQw4w9WgXcQ'
      }
}

    Añade un recurso para una cierta id de maquina.

    HTTP Request

    POST http://3.138.175.135:4000/Recursos/Add/

    Query Parameters

    Parametros Descripción
    tipo_maquina Refiere al tipo de maquina dado.
    url El URL del recurso.

    Maquina

    /Maquina/Search

    Utilizar Request para poder consumir la información

    
import requests

endpoint= 'Search'
url = f'http://3.138.175.135:4000/Maquina/{endpoint}'
response = requests.get(url, params={
    'tipo_maquina': '1',
})
print(response.json())

    Esta llamada entrega una respuesta del siguiente formato:

    {
    "Maquinas": [
        {
            "id_maquina": 1,
            "nombre": "algun_nombre",
            "tipo_maquina": "1",
            "ubicacion": "algun_lugar"
        },
        {
            "id_maquina": 3,
            "nombre": "xd",
            "tipo_maquina": "1",
            "ubicacion": "algun_lugar"
        }
    ]
}

    Devuelve todas las maquinas de un tipo

    HTTP Request

    GET http://3.138.175.135:4000/Maquina/Search

    Query Parameters

    Parametros Descripción
    tipo_maquina Si se añade permite filtrar por un tipo de maquina.
    Omitir un parametro omite el filtro.

    /Maquina/Add

    Utilizar Request para poder consumir la información

    
import requests

endpoint= 'Add'
url = f'http://3.138.175.135:4000/Maquina/{endpoint}'
response = requests.post(url, params={
    'tipo_maquina': '2',
    'nombre': 'maquina-chan',
    'ubicacion': 'algun_lugar',
})
print(response.json())

    Esta llamada entrega una respuesta del siguiente formato:

    {
    "respuesta": "OK",
    "maquina": {
        "id": 7,
        "tipo_maquina": "2",
        "nombre": "maquina-chan",
        "ubicacion": "algun_lugar",
        "id_maquina": 7
    }
}

    Crea una nueva maquina.

    HTTP Request

    GET http://3.138.175.135:4000/Maquina/Add

    Query Parameters

    Parametros Descripción
    tipo_maquina Refiere al tipo de la maquina.
    nombre Es el nombre de la maquina añadida.
    ubicación Ubicación de una maquina.

    Parametros Descripción

    Habilitación

    /Habilitacion/Search

    Utilizar Request para poder consumir la información

    
import requests

endpoint= 'Search'
url = f'http://3.138.175.135:4000/Habilitacion/{endpoint}'
response = requests.get(url, params={
    'id_maker': '1',
    'tipo_maquina': '1'
})
print(response.json())

    Esta llamada entrega una respuesta del siguiente formato:

    {
    "Estado": "Habilitado"
}

    Devuelve si un Maker está habilitado o no.

    HTTP Request

    GET http://3.138.175.135:4000/Habilitacion/Search

    Query Parameters

    Parametros Descripción
    id_maker El ID de un maker.
    tipo_maquina El tipo de una maquina dada.

    /Habilitacion/Add

    Utilizar Request para poder consumir la información

    
import requests

endpoint= 'Add'
url = f'http://3.138.175.135:4000/Habilitacion/{endpoint}'
response = requests.post(url, params={
    'id_maker': '2',
    'tipo_maquina': '2',
    'habilitado_o_no': 'y'
})
print(response.json())

    Esta llamada entrega una respuesta del siguiente formato:

    {
    "respuesta": "OK",
    "habilitacion": {
        "id": 4,
        "id_maker": "2",
        "tipo_maquina": "2",
        "habilitado_o_no": "y"
    }
}

    Almacena el Estado de un Usuario.

    HTTP Request

    POST http://3.138.175.135:4000/Habilitacion/Add

    Query Parameters

    Parametros Descripción
    id_maker El ID de un maker.
    tipo_maquina El tipo de una maquina dada.
    habilitado_o_no "y" en caso de estar habilitado, "n" en caso contrario.

    /Habilitacion/Update

    Utilizar Request para poder consumir la información

    
import requests

endpoint= 'Update'
url = f'http://3.138.175.135:4000/Habilitacion/{endpoint}'
response = requests.put(url, params={
    'id_maker': '2',
    'tipo_maquina': '2',
    'habilitado_o_no': 'n'
})
print(response.json())

    Esta llamada entrega una respuesta del siguiente formato:

    {
    "respuesta": "OK",
    "habilitacion": {
        "habilitado_o_no": "n",
        "id_maker": "2",
        "tipo_maquina": "2"
    }
}

    Actualiza un Estado de un Maker.

    HTTP Request

    PUT http://3.138.175.135:4000/Habilitacion/Update

    Query Parameters

    Parametros Descripción
    id_maker El ID de un maker.
    tipo_maquina El tipo de una maquina dada.
    habilitado_o_no "y" en caso de estar habilitado, "n" en caso contrario.

    Errors

    La API de Recursos presenta los siguientes manejos de errores:

    Error Code Meaning
    400 Bad Request -- Your request is invalid.
    404 Not Found -- The specified endpoint could not be found.
